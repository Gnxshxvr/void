# **App Name**: MatsyaSetu Marketplace

## Core Features:

- Multilingual Support: Offer the website content in Tamil, Hindi, Bengali, and English to cater to a diverse user base.
- Seller Fish Listing: Enable sellers to post fish listings with details such as name, weight, price, freshness, and image.
- Buyer Fish Marketplace: Enable buyers to view fish listings with searching and sorting
- Purchase and PDF bill generation: Allow buyers to 'purchase' items and generate a PDF bill with a calculated 2.5% commission + tax. No payment APIs.
- Authority Dashboard: Provide an admin dashboard for authorities to view total sales, profits, commissions.
- Graphical Analytics: Generate real-time graphical analytics (bar/line charts) for sales and commission data, displayed in the authority dashboard.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), evoking the ocean depths and the heart of the fishing industry.
- Background color: Very light blue (#F0F9FF), to suggest cleanliness.
- Accent color: Sea green (#34D399), drawing upon colors in the local Tamil environment for freshness.
- Body font: 'PT Sans', sans-serif, a humanist sans-serif that combines a modern look and a little warmth or personality, suitable for user interface text.
- Headline font: 'Playfair', a modern sans-serif similar to Didot, geometric, high contrast thin-thick lines, with an elegant, fashionable, high-end feel
- Employ fishing and coastal Tamil-themed icons throughout the platform to reinforce the marketplace's identity.
- Clean and lightweight dashboard UI with intuitive data visualization components, inspired by Tamil coastal styling, for the authority dashboard.
- Subtle transitions and animations to enhance user engagement and provide real-time updates on data.
